🌾 KisanSetu

KisanSetu is a modern digital platform that connects farmers with tractor drivers for agricultural services. It is built using Next.js, React, and Supabase, providing a fast, secure, and responsive experience.

---

✨ Features

- 🌾 Farmer Registration
- 🚜 Driver Registration
- 🔐 Secure Authentication
- 📲 OTP Login
- 🪪 Aadhaar Verification
- 📁 Document Upload
- 📅 Tractor Booking System
- 🚜 Driver Assignment
- 🔔 Real-time Notifications
- 💬 Driver & Customer Chat
- 📊 Admin Dashboard
- 💰 Earnings Management
- 📜 Booking History
- ☁️ Supabase Backend
- 📱 Fully Responsive Design
- 🚀 Vercel Deployment Ready

---

🛠️ Tech Stack

- Next.js
- React.js
- JavaScript (ES6+)
- Supabase
- PostgreSQL
- Vercel

⚙️ Installation

Clone the repository:

git clone https://github.com/your-username/Kisansetu2026.git

Install dependencies:

npm install

Run the development server:

npm run dev

Create a production build:

npm run build

Run the production server:

npm start

---

🔑 Environment Variables

Create a file named:

.env.local

Add the following variables:

NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
